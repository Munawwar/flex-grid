Dependency

grid.less depends on flex-helper-2012.css from https://github.com/Munawwar/flex-helper

Class Names

Naming convention is similar to bootstrap 3 but with a 'f-' prefix.

Just like bootstrap the CSS priority is "mobile first". i.e. if you apply f-col-xs-12, on an element, then it applies to all device widths unless another higher-width CSS is applied (like f-col-md-*).
